## 2026-01-24 - Micro-UX: Empty State & ARIA Labels in Ingredients List\n**Learning:** Users perceive "broken" features when lists are empty without guidance. Adding a dedicated empty state with an icon and CTA significantly improves perceived polish. Also, icon-only buttons are invisible to screen readers without `aria-label`.\n**Action:** Always check list components for empty state handling and icon buttons for accessibility labels.
